
If you are using an older version of Videomass lower than version 5.0.5 the 
presets in this folder will not be compatible. To make them compatible you 
will have to rename the file extensions from prst to json .

Conversely, on a new version of Videomass, in order to use a preset that 
you saved using an older version of Videomass, you will need to rename the 
extension from prst to json .
